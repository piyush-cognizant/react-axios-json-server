## Clone repo
```bash
git clone https://github.com/piyush-cognizant/react-axios-json-server.git
```

## Change Directory
```bash
cd react-axios-json-server
```

## Install Dependencies
```bash
npm install
```

## Install json-server globally
```bash
npm install -g json-server
```
## Run the json-server
```bash
json-server --watch books.json --port 4000
```

## Run the react app (in seperate terminal)
```bash
npm run dev
```
